## Scripts

To run the file, please make sure you first `npm install` all of the dependencies required for this project.

- `npm run start-client` will have webpack build the front-end (client) side of the application
- `npm run server_start` will launch the NodeJS (Express) server which will also serve the static files for the project to work.

The port number will be your http://localhost:3000/, which will server the files to your local machine when 
run in the correct environment.

Ensure you have a code editor of your choice available, along with npm and node installed.
